# Setup Bitfusion Client for Ubuntu18.04
## Install Git
---
```
$ sudo apt update
$ sudo apt -y install git
```

## Clone repogitory
---
```
$ cd && git clone https://github.com/y-yas/bf/
```

## Download Bitfusion client
---
```
$ ./setup_ubuntu.sh
```

## Enable Bitfusion client 
---
vCenterサーバでクライアントのEnableを実施
